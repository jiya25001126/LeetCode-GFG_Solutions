# [nPr](https://www.geeksforgeeks.org/problems/npr4253/1)
## Easy
Write a program to calculate nPr .&nbsp;nPr &nbsp;represents n&nbsp;permutation&nbsp;r and value of&nbsp;nPr &nbsp;is (n!) / (n-r)!.
Examples:
Input: n = 5, r = 2
Output: 20
Explaination: 5!/(5-2)! = 5!/3! = 120/6 = 20.
Input: n = 6, r = 3
Output: 120
Explaination: 6!/(6-3)! = 6!/3! = 720/6 = 120.
Constraints:1 ≤ n, r ≤ 20
