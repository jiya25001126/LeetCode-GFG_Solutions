# [Divisible by 13](https://www.geeksforgeeks.org/problems/divisible-by-13/1)
## Medium
Given a number represented as a string s&nbsp;(which may be very large), check whether it is divisible by 13 or not.
Examples:
Input : s = "2911285"Output : trueExplanation: 2911285 / 13 = 223945, which is a whole number with no remainder.
Input : s = "27"Output : falseExplanation: 27 / 13 ≈ 2.0769..., which is not a whole number (there is a remainder).
Constraints:1 ≤&nbsp; s.size()&nbsp; ≤ 105
